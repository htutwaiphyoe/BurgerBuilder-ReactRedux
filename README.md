# Burger Builder

## Installation

-   download the project folder
-   npm install for node_modules
-   npm start for development mode

## Technologies

-   React
-   Redux
-   Firebase
-   React-Router-Dom
-   React-Redux
-   Redux-Thunk
-   Redux-Dev-Tool
-   Axios

## Features

-   Burger Builder
-   Order Form
-   Orders
-   Authentication
-   Testing
